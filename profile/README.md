Je me presente je m'appelle Souilah Walid, actuellement je suis dans la L2 Informatique et je suis le createur de ce github. Le but de ce github est simple,
c'est de pouvoir aider les personnes qui rejoint la licence en leur donnant tout les informations necessaire et outils a disposition. Ce github regroupe un peu
tout ce que les gens qui etait avant moi m'ont donne et ce que moi de mon cote j'ai fait au moins ici vous pouvez retrouver la plupart des controls ou exercices

Ce github est ici pour etre un plus pas pour remplacer tout sorte d'aide ou information.
